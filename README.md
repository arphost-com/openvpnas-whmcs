# OpenVPNAS WHMCS Module

## Admin Features

- Deploys single user on order. (First payment of the option you choose in the module)
- Administrator can change password in WHMCS.

## Client Features

- Download profile.
- Change WHMCS password.
- Link to OpenVPN Access Server Portal.

## Requirements

- Docker
- OpenVPN Access Server Docker Container.

There are a lot of ways this can be setup. Yo can setup a reverse proxy for the web interface and. firewall for both only allowing VPN ports and Secure Web Access. If you need help contact us at https://arphost.com
